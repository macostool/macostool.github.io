"Open" the directory where the video file is located.

Can contain video files in its subdirectories.

Method 1: Click directly on the video to select a certain frame of image as the cover of the video file.

Method 2: Set an integer for each video or an image that counts down a certain moment, and click "Apply to All" to add cover to video files in batches.

Note: If the set moment exceeds the length of the video, the image from the last moment will be used.
